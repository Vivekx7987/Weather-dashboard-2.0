# SkySense - Weather Dashboard 🌤️

SkySense is a modern weather dashboard that provides real-time weather data with dynamic UI, animations, and recommendations.

## Features
- 🌍 Search weather by city
- 📍 Get weather using current location
- 🌡️ Real-time temperature and weather data
- 🎨 Dynamic UI that changes with weather
- 🧥 Outfit recommendations
- 🎯 Activity suggestions
- 🎬 Hero video landing section
- 🌙 Dark / Light mode
- 📱 Fully responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API
